Functions in JavaScript

Check the console for outputs of the functions.

Functions are reusable blocks of code that perform a specific task.

They can take inputs (parameters) and return outputs.

Here are some examples of functions:

  • add7: Adds 7 to a number.
  • multiply: Multiplies two numbers.
  • capitalize: Capitalizes the first letter of a string.
  • lastLetter: Returns the last letter of a string.

Open the console to see the results of these functions in action.

Note: The functions are defined using both traditional function syntax and arrow function syntax.

Feel free to modify the inputs and see how the outputs change!

Enjoy learning about functions in JavaScript!
